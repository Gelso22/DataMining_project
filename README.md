# Stock Performance Classification

Early data science project focused on stock performance prediction using company financial metrics.

The objective of this project was to build a binary classification model to predict whether a stock would have positive or negative performance in the following year. The analysis was based on a ready-to-use Kaggle dataset containing financial indicators for NYSE-listed companies.

## Project overview

The project includes:

- exploratory data analysis on financial metrics and stock performance;
- preprocessing of missing values, zero values, redundant variables, and class imbalance;
- feature selection and feature engineering;
- comparison between logistic regression and k-nearest neighbors;
- model evaluation using accuracy, ROC AUC, sensitivity, and specificity.

The final selected model was a logistic regression model combined with upsampling of the minority class.
